# Publish This Bundle

Use this file from inside `dist/plant-decor-selector-pack/` during the one required publication step.

## Create The Repo

- Repository name: `plant-decor-selector-pack`
- Visibility: `Public`
- Description: `USD 12 AI-assisted indoor plant decor selector pack for one room or corner.`

## Upload This Folder

Upload the full contents of this folder to the new repository root, including:

- `README.md`
- `docs/`
- `offers/`

## Enable Pages

- Open repository `Settings`
- Open `Pages`
- Source: `Deploy from a branch`
- Branch: `main`
- Folder: `/docs`
- Save

## Set The Repo Website Link

- Wait for the Pages URL to appear in GitHub
- Open the repo main page
- In the right-side `About` box, edit `Website`
- Paste the live GitHub Pages URL
- Save

## Check Before Leaving

1. Confirm the README reads clearly on GitHub.
2. Confirm the repo `About` website link points to the live GitHub Pages URL.
3. Open the GitHub Pages URL and confirm the offer page loads.
4. Confirm the sample and intake links work.

- Repository URL
